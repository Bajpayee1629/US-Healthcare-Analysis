# US Healthcare Data Analysis

### 📌 Business Problem
Analyzed 10,000 US healthcare patient records to identify impact of demographics & admission types on costs and hospital resource utilization.

### 🛠️ Tools:  Excel, Pivot Table, Power Query, PowerPoint

### 📊 Key Insights
- **Demographics**: 60% patients are 50+ years old. Arthritis dominates 65+ age group
- **Cost Analysis**: Emergency admissions cost 1.4x more than Elective. 65% patients in High/Very High cost bucket
- **Resource Management**: Elective procedures avg 12+ days LOS, blocking bed turnover
- **Business Impact**: Potential $12M+ annual savings via targeted interventions

### 🔄 My Process
1. **Data Cleaning**: Removed 11 empty columns, standardized formats, handled 3,011 invalid date records
2. **Feature Engineering**: Created MECE Age Buckets & Cost Buckets for analysis
3. **Analysis**: Used Pivot Tables to find trends across demographics, billing, LOS
4. **Visualization**: Built 11-slide PowerPoint with 8 charts for stakeholder presentation
